While the preview is running, let us examine the gadget code in versal.html. It is self-contained and consists of a simple CSS style sheet and a small JavaScript snippet:

```javascript

greeting.addEventListener(‘change’, function(e){ player.setAttribute(greeting.name, greeting.value); });

learnerName.addEventListener(‘change’, function(e){ player.setLearnerAttribute(learnerName.name, learnerName.value); });

player.on(‘attributesChanged’, function(data){ if(data.greeting) { greeting.value = data.greeting; }; });

player.on(‘learnerStateChanged’, function(data){ if(data.learnerName) { learnerName.value = data.learnerName; }; });

player.on(‘editableChanged’, function(editable){ greeting.readOnly = !editable.editable; learnerName.readOnly = editable.editable; });

player.setHeight(60); player.startListening(); </script> ```

Let’s set a background color on the gadget. Add these lines to the <style> tag in versal.html:

  body {
    background-color: #def;
  }
